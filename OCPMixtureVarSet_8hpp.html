Detailed Description

OCPMixtureVarSet类的声明

Author
Shizhe Li
Date
Oct/02/2023

OCPMixtureVarSet类用于表示和初始化流体混合物的各种属性和状态。 它包括了相的类型、相的存在性、各相的物性数据以及它们的导数等。 本文件还定义了相关的枚举类型以表示不同的混合物类型和相类型。
